## How to Run

1. Clone this repository at https://github.com/dayanelaura/track-it.git
2. Install the dependencies.
```bash
npm i
```
3. Execute
```bash
npm run start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
